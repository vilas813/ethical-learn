1️⃣ Nmap के 30 Powerful Commands

Basic Scan

nmap target.com

IP Scan

nmap 192.168.1.1

Multiple IP Scan

nmap 192.168.1.1 192.168.1.2

Network Scan

nmap 192.168.1.0/24

Ping Scan

nmap -sn 192.168.1.0/24

Specific Port Scan

nmap -p 80 target.com

Multiple Ports Scan

nmap -p 80,443 target.com

Port Range Scan

nmap -p 1-1000 target.com

Fast Scan

nmap -F target.com

Service Version Detection

nmap -sV target.com

OS Detection

sudo nmap -O target.com

Aggressive Scan

nmap -A target.com

TCP SYN Scan

sudo nmap -sS target.com

UDP Scan

sudo nmap -sU target.com

Stealth Scan

sudo nmap -sS target.com

Firewall Detection

nmap -sA target.com

Top Ports Scan

nmap --top-ports 20 target.com

Script Scan

nmap --script vuln target.com

HTTP Enumeration

nmap --script http-enum target.com

FTP Anonymous Check

nmap --script ftp-anon target.com

SMB Scan

nmap --script smb-os-discovery target.com

DNS Brute Force

nmap --script dns-brute target.com

Whois Info

nmap --script whois target.com

Traceroute

nmap --traceroute target.com

Output Save

nmap -oN result.txt target.com

nmap -oX result.xml target.com

Scan Speed Fast

Disable Ping

Detect All Ports

Combined Scan

nmap -A -T4 target.com

2️⃣ Website Hacking में Nmap का Use

Ethical Hackers Nmap का use Information Gathering के लिए करते हैं。

Step 1 – Target Identify

सबसे पहले website का IP पता करते हैं。

nmap target.com

Step 2 – Open Ports Find

nmap target.com

Example Output:

80/tcp open http 443/tcp open https 22/tcp open ssh

इससे पता चलता है कि server पर HTTP, HTTPS और SSH open है।

Step 3 – Services Identify

nmap -sV target.com

Example:

Apache 2.4.49 OpenSSH 7.6

अब hacker को पता चलता है कि server पर कौन-सा software version चल रहा है।

Step 4 – Vulnerability Scan

nmap --script vuln target.com

इससे server की known vulnerabilities पता चल सकती हैं।

3️⃣ Nmap Practical Example (Step-by-Step)

मान लो target website है:

example.com

Step 1 – Basic Scan

nmap example.com

Step 2 – Service Detection

nmap -sV example.com

Step 3 – OS Detection

sudo nmap -O example.com

Step 4 – Vulnerability Scan

nmap --script vuln example.com

Step 5 – Aggressive Scan

nmap -A example.com

इससे hacker को मिल सकता है:

  • Open Ports
  • Server Software
  • Operating System
  • Possible Vulnerabilities

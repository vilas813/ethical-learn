1️⃣ Nmap क्या है

Nmap (Network Mapper) एक बहुत famous network scanning और security auditing tool है। इसका उपयोग ethical hackers और security researchers करते हैं。

Nmap से आप ये काम कर सकते हो:

  • Network में live devices (hosts) पता करना
  • Open ports check करना
  • Server पर कौन-सी services चल रही हैं देखना
  • Operating System detect करना
  • Network security vulnerability analysis करना

Example: अगर किसी website/server पर कौन-सा port खुला है यह जानना हो तो Nmap use किया जाता है।

2️⃣ Kali NetHunter में Installation

Kali NetHunter में Nmap usually पहले से installed होता है।

अगर नहीं है तो install करने के लिए:

apt update
apt install nmap

Install check करने के लिए:

nmap --version

3️⃣ Basic Usage Commands

1. Simple Scan

nmap example.com

यह target website के open ports scan करेगा।

2. IP Range Scan

nmap 192.168.1.1-100

यह 1 से 100 तक के devices scan करेगा।

3. Specific Port Scan

nmap -p 80 example.com

यह सिर्फ port 80 scan करेगा।

4. OS Detection

nmap -O example.com

यह server का operating system detect करने की कोशिश करेगा।

5. Service Version Detection

nmap -sV example.com

यह बताएगा कि server पर कौन-सी service और version चल रहा है।

💡 Pro Tip (Ethical Hacking)

Nmap का use हमेशा legal penetration testing या learning lab में ही करना चाहिए।

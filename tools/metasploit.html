Metasploit क्या है?

Metasploit एक open-source penetration testing framework है जो security vulnerabilities को exploit करने के लिए इस्तेमाल होता है। यह tool exploits, payloads, और auxiliary modules provide करता है।

Metasploit का उपयोग

Metasploit का उपयोग vulnerability exploitation में होता है। यह remote code execution, privilege escalation, और post-exploitation tasks के लिए इस्तेमाल होता है।

Basic Commands

  • msfconsole - Metasploit console start करना
  • search exploit_name - Exploit search करना
  • use exploit/path - Exploit select करना
  • show options - Options देखना
  • exploit - Exploit run करना

Kali Linux में Install करना

Metasploit by default Kali Linux में installed होता है। अगर नहीं है तो:

sudo apt update && sudo apt install metasploit-framework

Important Note

इस tool का उपयोग केवल authorized systems पर करें। Unauthorized exploitation illegal है।

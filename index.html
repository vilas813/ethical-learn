Cyber Security और Ethical Hacking सीखें

Ethical Learn पर आसान हिंदी भाषा में Kali Linux tools, hacking concepts और security techniques के बारे में जानें। अपने systems को सुरक्षित रखने के लिए cyber security को समझें।

Tools देखें

1️⃣ Information Gathering Tools

Target website या network की information collect करने के लिए इस्तेमाल होते हैं। Domain, IP address, subdomain, email, server details जैसी जानकारी निकालते हैं। Penetration testing का पहला step होता है。

  • Nmap
  • Maltego
  • theHarvester
  • Recon-ng
  • dnsenum
  • dnsrecon

2️⃣ Vulnerability Analysis

Website या system में security vulnerabilities खोजते हैं। जैसे outdated software, open ports, SQL injection जैसी कमजोरियाँ। Hacker या security tester को attack से पहले weaknesses पता चलती हैं。

  • Nikto
  • OpenVAS
  • OWASP ZAP
  • SQLmap

3️⃣ Web Application Attacks

Website और web apps की security testing के लिए इस्तेमाल होते हैं। Login system, forms, cookies, sessions आदि को test करते हैं। Web vulnerabilities exploit करने के लिए उपयोग होते हैं。

  • Burp Suite
  • Commix
  • Skipfish
  • Wfuzz

4️⃣ Password Attacks

Password को brute force या dictionary attack से crack करने के लिए। Weak passwords को identify करने में मदद करते हैं। Security testing में authentication system check करने के लिए उपयोग।

  • Hydra
  • Hashcat
  • John the Ripper
  • Medusa

5️⃣ Wireless Attacks

WiFi networks की security testing के लिए। WPA/WPA2 passwords crack करना, WPS attack करना। Wireless network की vulnerabilities check करने के लिए।

  • Aircrack-ng
  • Reaver
  • Kismet
  • Wifite

6️⃣ Exploitation Tools

Vulnerability मिलने के बाद exploit करके system access लेने के लिए। Remote shell, payloads, backdoor create कर सकते हैं। Penetration testing में बहुत important stage।

  • Metasploit
  • BeEF
  • Armitage

7️⃣ Sniffing & Spoofing

Network traffic को capture और analyze करने के लिए। Data packets, passwords, sessions intercept कर सकते हैं। Man-in-the-Middle attacks के लिए इस्तेमाल होते हैं।

  • Wireshark
  • Ettercap
  • Bettercap

8️⃣ Forensics Tools

Cyber attack के बाद digital evidence analyze करने के लिए। Deleted files recover करना, malware analyze करना। Cyber crime investigation में इस्तेमाल।

  • Autopsy
  • Binwalk
  • Foremost
